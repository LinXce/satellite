[STS2 mod] 储君辉星旋转mod

本质自用mod，贴合Mesugaki_Regent储君娘化做的。
如果倾斜角度不对就该这两个变量吧：
src\RegentSatelliteOrbit.cs
			float verticalScale = 1.2f;
			float horizontalShear = 0.13f;
添加了一些美化，现在储君的星星会显示出来。
很简陋的mod，因为就做了两个小时。并且有大佬的美化mod做的非常好。

# mod使用方法

1. 下载 [Releases] 的压缩包 解压到杀戮尖塔2游戏目录下的mods文件夹。
2. 如果没有mods文件夹可以手动创建。
